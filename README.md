# **Mobile Store - MERN Project**

Welcome to **Mobile Store**! This project is a complete e-commerce application for mobile devices, developed using the **MERN** stack (MongoDB, Express.js, React, and Node.js).

## **Features**

- **User Authentication**: Sign up and log in  
- **Product Management**: Create, read, update, and delete products  
- **Shopping Cart**: Add and remove products  
- **Order Processing**: Simulated payments and inventory control  
- **Admin Panel**: Manage users and products  
- **Mobile-Optimized User Interface**  
---

## **Technologies Used**

### **Frontend**
- **React** with **Hooks** and **Context API** for global state management  
- **React Router** for application routing  
- **Material UI** for responsive design  

### **Backend**
- **Node.js** with **Express.js** as the framework  
- **MongoDB** as the NoSQL database  
- **Mongoose** for data modeling  
- **JSON Web Tokens (JWT)** for authentication
